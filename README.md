## Project Overview
This repository hosts the curated dataset (playlists and channel metadata) for under active development application.

## About the Application
The application is an educational tool designed to organize YouTube playlists into structured courses (Programming, Design, etc.). It utilizes the **YouTube Data API** to fetch video details and displays them using the standard **YouTube IFrame Player**.

**Current Status:**
- The application source code is currently **private** and under active development.
- This repository serves as the public data source ensuring transparency in how content is curated.
- The application strictly adheres to YouTube's Developer Policies and Terms of Service.
